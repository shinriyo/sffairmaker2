動作環境
---------
Windows 7で、それなりのメモリ・ハードディスクを持った環境。
確認していませんが、VistaやXPでも多分動くと思います。

インストール
-------------
特別な作業は必要ありません。
適当なフォルダにファイルを解凍し、SFFAIRMaker.exeを実行してください。

アンインストール
-----------------
単に削除してください。
設定ファイルが%APPDATA%\SFFAIRMaker2に作られますが、
ごく小さいので削除しなくても問題ないでしょう。

これは仕様です
--------------
* AIRのコメントが消える
* 新規作成したり画像を全部削除したりすると、変な画像が追加されたり1枚だけ画像が残ったりする
* MCMとパレット共有のチェックが違う(MCMとは考え方が違います)
* SFFのサイズが（たまに）デカイ
* 起動が遅い
